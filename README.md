Consumer Trends Analysis

Overview

This project analyzes customer purchasing behavior and consumer trends using **Python**, **PostgreSQL (SQL)**, and **Power BI**. It showcases an end-to-end analytics workflow from data cleaning to insight generation and dashboard visualization—to support data-driven business decisions.


Objectives

* Understand customer spending patterns
* Segment customers (New, Returning, Loyal)
* Analyze product performance and trends
* Evaluate impact of discounts and subscriptions


Tech Stack

* Python (Pandas, NumPy) – Data preprocessing
* PostgreSQL (SQL) – Data analysis
* Power BI – Dashboard & visualization



Workflow

1. Data Cleaning (Python)

* Handled missing values and inconsistencies
* Standardized data formats and categories
* Prepared dataset for analysis

2. Data Analysis (SQL)

* Used CTEs, window functions, subqueries, and aggregations
* Solved key business problems like:

  * Revenue by gender
  * Discount vs spending behavior
  * Subscription impact on revenue
  * Top products by rating and sales
  * Customer segmentation & retention

3. Dashboard (Power BI)

* Built an interactive dashboard with:

  * KPIs (Revenue, Customers, Avg Spend)
  * Customer segments
  * Product & category insights
  * Demographic analysis

Key Insights

* Loyal customers drive a major share of revenue
* Discounts can attract high-spending customers
* Subscribed users contribute more to total revenue
* Clear trends across age groups and product categories


Skills

Python • SQL • Data Analysis • Power BI • Data Storytelling
